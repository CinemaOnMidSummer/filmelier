HTML 에서 요소의 'id'값으로 사용되는 문자열은 숫자로만 이루어질 수 없다고 합니다. 복잡한 코드를 작성해야할 수 있는 상황에서 숫자를 문자열로 변환하여 'id'로 사용하면 코드를 보다 직관적으로 만들 수 있기 때문이라네요.. (출처: chat gpt, 반박시 님 말 맞음)

function getRandomTile(){
// math.random --> (0-1) * 9 = (0-9) -->round down to (0-8) integers
let num = Math.floor(Math.random() * 9);
return num.toString();
}

줌 검색등록
검색 사이트->줌 검색등록 검색->[신규 등록] 클릭 -> 개인정보 제공 동의->블로그에 관한 정보 간단히 입력하면 신청 완료
이용방법 안내 사이트 등록/수정/삭제 신청 절차에 대해 자세히 안내 해 드립니다. 등록가이드 사이트 검색등록 기준 및 제출 서류 안내, 작성가이드 등을 안내 해 드립니다. 자주묻는 질문 사이트 검색등록 시 자주 묻는 질문에 대해 상세한 답변을 드립니다.

구글 검색등록(키자드)
검색 사이트->기자 드 검색->접속 후 회원가입 ->로그인->[네이버 블로그 구글 검색등록]에서 검색될 포스트의 주소를 복사해 붙여 넣은 후 [검색등록] 클릭
* 키자드에서 네이버 블로그를 등록하는 방법 세 가지
네이버 블로그 포스트 주소 입력 : 네이버 ID 연동 이후 포스트 주소를 개별적으로 입력하여 등록하는 방식, 하루에 한 개씩 무료로 이용
RSS 등록 : 1일 1회 무료 이용 가능, 최근 50개의 포스팅 URL을 한 번에 등록 가능
전체 등록 : 공개된 모든 포스트를 한 번에 등록 가능하지만 일정 포인트가 필요, 포인트 획득하는 방법으로는 1일 1 로그인 출석, 키자드 사이트에 글쓰기, 댓글 쓰기, 게시글 추천 등
네이버 블로그를 구글 검색에 등록하는 서비스. 구글 검색 노출 및 백링크를 걸어서 SEO최적화를 시켜줍니다.
