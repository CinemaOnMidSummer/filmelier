# filmelier

맞춤형 알고리즘 영화관에 오신 여러분을 환영합니다!
사용자의 취향을 고려할 수 있는 맞춤형 영화 추천 서비스를 제공합니다.
궁금한 영화 제목을 검색하실 수 있습니다.
접속 시 포스터가 등장합니다.
포스터에 마우스를 올리면 1점부터 5점까지 별점을 남기실 수 있습니다.
이후 '영화 추천받기'라는 버튼을 클릭하시면 유사한 영화가 추천됩니다.
순위에 따른 영화를 제공받을 수 있습니다.
다양한 영화를 마음껏 누려 보세요!
